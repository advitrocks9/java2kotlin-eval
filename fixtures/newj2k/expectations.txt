# Hypothesis checks for fixtures/newj2k -- the 15-pair authentic J2K
# input/output sample from intellij-community's own testData (pinned commit,
# fetched by scripts/fetch-newj2k-fixtures.sh). The .kt side of each pair is

# Format: relpath | tag | yes|no | regex | description

# Functional interfaces. Annotated SAM gets `fun interface` lift. The
# unannotated case does NOT promote -- J2K is conservative without the
# @FunctionalInterface annotation. Important: this contradicts an earlier
# claim in docs/EDGE_CASES.md that "J2K detects the structural shape, not
# just @FunctionalInterface" -- the actual fixture proves otherwise. Doc is
# corrected; this expectation is the executable proof.
functionalInterfaces/MyRunnable/MyRunnable.kt                                           | fun_interface_promoted                       | yes | fun interface MyRunnable | @FunctionalInterface annotated SAM becomes fun interface
functionalInterfaces/NoFunctionalInterfaceAnnotation/NoFunctionalInterfaceAnnotation.kt | fun_interface_NOT_promoted_without_annotation | no  | fun\s+interface          | structural SAM without @FunctionalInterface stays plain interface
